SIRAAS

SIRAAS é um site de gerenciamento de atestados médicos e times scrum, realizado no 1º semestre da faculdade de ADS como projeto avaliativo.

O projeto inclui um sistema de upload de atestados, gerenciamento deles por status de aprovação, visualização gráfica de afastamentos, registros de equipes Scrum, sistema de avaliação de membros da equipe e visualização gráfica das avaliações.

Minha participação no projeto

  • HTML e CSS da página do docente
  • HTML e CSS da página de estatísticas de afastamento
  • HTML da página de cadastro dos membros Scrum
  • Back-end do cadastro com Flask gravando as informações em um arquivo texto (.txt)
  • Modelagem dos casos de uso no Astah

Desafios enfrentados

Esse projeto foi minha primeira experiência com programação. Comecei a faculdade sem nenhum conhecimento na área. Foi um projeto de aprendizado onde tive que me adaptar a metodologia Scrum e a programação.
